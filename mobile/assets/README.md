# Assets Klasörü

Bu klasöre icon dosyaları gelecek ama şimdilik boş bırakabilirsiniz.

Gerekirse şu komutlarla boş dosyalar oluşturun:

```bash
# Windows PowerShell:
cd assets
echo. > icon.png
echo. > adaptive-icon.png  
echo. > splash.png
echo. > favicon.png

# Linux/Mac:
touch icon.png adaptive-icon.png splash.png favicon.png
```

Veya iconlar olmadan da çalışır, Expo varsayılan iconları kullanır.
